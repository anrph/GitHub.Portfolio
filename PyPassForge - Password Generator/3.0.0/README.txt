----- OVERVIEW -----
* This Python script generates strong and secure passwords, it's a simple yet effective tool for creating unique 
  passwords for ensuring enhanced security for your accounts and data.

----- REQUIREMENTS -----
* Python 3.x
* No additional dependencies required

----- SET UP -----
1. Clone or download the PyPassForge.py script to your local machine.
2. Open a terminal or command prompt.
3. Navigate to the directory where PyPassForge.py is located.
4. Run the script by executing the following command:
  python PyPassForge.py

----- USAGE -----
* [ 1, 2, 3, 4]  Will turn on & off the four main character groups from the password generation.
* [ 1011 ] Four digit binary prompts will turn on/off multiple character groups at once (0 = off, 1 = on).
* [ 8 : 30 ] Numbers from 8 to 30 for changing the password length.
* [ r(x..) ] Use the letter "r" followed by the specific characters to remove from the generation algorithm.
* [ a(x..) ] Use the letter "a" followed by the specific characters to add back to the generation algorithm.
* [ r ] Print the removed characters list.
* [ R ] Reset the removed characters list.
* [ s ] Save a generated password using the letter "s".
* [ q ] Exit the program using the letter "q".
